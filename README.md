# Exploring Factors Influencing Fertility Rates

## Project Overview
Analysis of fertility rates across Asian countries examining economic, educational, health, and environmental factors using Tableau.

## Visualizations Included
- **Geographic Map**: Fertility rate distribution across Asia
- **Dumbbell Chart**: Fertility rate changes from 2000s to 2010s
- **Scatter Plot**: Literacy rate vs fertility rate correlation
- **Line Graph**: GDP per capita vs fertility rate trends
- **Dual Bar Chart**: Fertility rate vs life expectancy comparison

## Tools Used
- Tableau

## Key Insights
- Negative correlation between literacy rates and fertility rates
- Higher GDP per capita associated with lower fertility rates
- Countries with higher fertility rates tend to have lower life expectancy

## How to View
Download the .twbx file and open with Tableau Public or Tableau Desktop.
